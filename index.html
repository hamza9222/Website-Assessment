Cube Spinner

A REVOLUTIONARY AND WORLD CHANGING GAME

The process of making this game included creating a new 3D project in Unity. After that, I created a 3D plane asset which acted as the 'ground' in the game. After that, I created a 3D cube asset which will be main game object. Then, I created unique materials for both the objects and applied it onto them. After that, I created a new script called "CubeSpinner" and wrote some code and magic to make the game work. And that's how I made my game. Voila!
